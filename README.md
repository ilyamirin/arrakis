# Arrakis Spice Harvest

Игра в сеттинге вселенной Dune, где игрок управляет харвестером для добычи спайса на планете Арракис.

## Описание игры

- **Цель**: Собрать весь спайс (25 единиц) на поле 9x9, избегая песчаных червей
- **Управление**: Харвестер движется буквой "Г" как шахматный конь
- **Опасность**: После каждого хода появляется червь на случайной свободной клетке
- **Победа**: Собрать весь спайс
- **Поражение**: Попасть на клетку с червем

## Установка и запуск

### Требования
- Python 3.x
- TypeScript (установить глобально: `npm install -g typescript`)

### Быстрый запуск
```bash
make run
```

### Пошаговый запуск
```bash
# 1. Установка зависимостей
make install

# 2. Сборка
make build

# 3. Запуск сервера
make serve
```

### Ручной запуск
```bash
# Компиляция TypeScript
tsc

# Запуск сервера
python3 server.py
```

## Игровые механики

- **Поле**: 9x9 клеток с ~25 единицами спайса
- **Харвестер**: Начинает в центре, движется буквой "Г"
- **Черви**: Появляются случайно после каждого хода
- **Спайс**: Автоматически собирается при входе на клетку
- **Вероятность победы**: ~50% при случайной игре

## Технологии

- **Frontend**: HTML5, CSS3, TypeScript
- **Графика**: Canvas 2D API
- **Сервер**: Python http.server
- **Сборка**: TypeScript Compiler (tsc)
