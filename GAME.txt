✦ Amber Dunes Harvest

    Актуальное описание:
    Это 100% AI-generated retro-futurist desert game. Игрок управляет Collector
    в регионе The Amber Waste, собирает весь amber на поле 8x8 и старается
    пережить случайные появления Sinkjaw.

    Текущие правила:

    1. Поле:
     - Размер поля: 8x8 клеток
     - На старте случайно раскладываются ровно 20 залежей amber
     - Центральная клетка остаётся свободной для старта Collector

    2. Collector:
     - Начинает в центре поля
     - Ходит буквой "Г", как шахматный конь
     - Может приземляться на пустую клетку или на клетку с amber
     - Если приземляется на amber, ресурс собирается автоматически

    3. Sinkjaw:
     - После каждого хода Collector выбирается новая клетка появления
     - Sinkjaw появляется случайно в радиусе до 4 клеток от Collector
     - Первые 3 хода Sinkjaw не может появиться прямо под Collector
     - Если новая клетка совпала с позицией Collector, игра проиграна немедленно

    4. Победа и поражение:
     - Победа: собран весь amber
     - Поражение: Sinkjaw появился на Collector
     - Поражение: после хода не осталось допустимых прыжков

    5. Интерфейс:
     - Игровая сцена рисуется на Canvas 2D
     - Вокруг canvas используется HTML/CSS интерфейс
     - Есть счётчики amber, ходов и текущей позиции
     - Есть статусная панель, легенда, Field guide и author block

    Технологии:
     - Frontend: HTML, CSS, TypeScript
     - Рендеринг: Canvas 2D API
     - Backend: отсутствует
     - Локальный запуск: Python HTTP server
