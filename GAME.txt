✦ Описание игры "Amber Dunes Harvest"

    Общее описание:
    Игра выполнена в сеттинге retro-futurist desert game, где игрок управляет Collector, устройством для добычи ценного
    amber в регионе The Amber Waste. Цель игрока - собрать весь amber с ограниченной области (доски 9x9), при этом избегать
    опасностей, особенно Sinkjaw, который охотится за Collector.

    Основные механики:

    1. Поле:
     - Игровое поле размером 9x9 клеток
     - Приблизительно 25 клеток содержат ценный amber (желтые клетки)
     - Каждый ход Sinkjaw может появиться на любой свободной клетке

    2. Collector:
     - Начинает в центральной клетке доски
     - Перемещается буквой "Г", как шахматный конь (например, 2 вперед + 1 вправо/влево, или 1 вперед + 2 вправо/влево)
     - Может прыгать только на пустые или содержащие amber клетки
     - При входе на клетку с amber - автоматически собирает его

    3. Sinkjaw:
     - После каждого хода игрока появляется на одной из случайных свободных клеток
     - Не может появиться на клетке, занятой Collector, или на той же клетке, где уже находится
     - Если Sinkjaw появляется на той же клетке, где находится Collector - игра проиграна

    4. Победа и поражение:
     - Победа: когда собран весь amber на поле
     - Поражение: когда игрок попадает в клетку, на которой появляется Sinkjaw
     - У среднего игрока вероятность выиграть составляет приблизительно 50% при случайных действиях

    5. Счетчики:
     - Количество собранных единиц amber (из общего количества)
     - Число совершенных ходов

    Технические детали:
     - Используется Canvas 2D API для визуализации
     - Написана на TypeScript с конвертацией в JavaScript
     - Для запуска используется Python HTTP-сервер
     - HTML пользовательский интерфейс с индикаторами состояния
     - Использует объектно-ориентированный подход для реализации игровых сущностей
     - Модульная структура: отдельные файлы для сущностей, игровой логики, отображения, утилит и основного скрипта

    Используемые технологии:
     - Frontend: HTML, CSS, TypeScript
     - Графика: Canvas 2D API
     - Backend: нет (чисто клиентское приложение)
     - Сервер: Python http.server для локального запуска
     - Система сборки: компиляция TypeScript -> JavaScript с помощью tsc (TypeScript Compiler)
     - Визуализация: 2D рендеринг на canvas с использованием ретро-футуристичной пустынной палитры (коричневый, золотой, бежевые оттенки)
