✦ Описание игры "Arrakis Spice Harvest"

    Общее описание:
    Игра выполнена в сеттинге вселенной Dune, где игрок управляет харвестером, специальным устройством для добычи ценного
    спайса на планете Арракис. Цель игрока - собрать весь спайс с ограниченной области (доски 9x9), при этом избегать
    опасностей, особенно песчаных червей, которые охотятся за харвестером.

    Основные механики:

    1. Поле:
     - Игровое поле размером 9x9 клеток
     - Приблизительно 25 клеток содержат ценный спайс (желтые клетки)
     - Каждый ход червь может появиться на любой свободной клетке

    2. Харвестер:
     - Начинает в центральной клетке доски
     - Перемещается буквой "Г", как шахматный конь (например, 2 вперед + 1 вправо/влево, или 1 вперед + 2 вправо/влево)
     - Может прыгать только на пустые или содержащие спайс клетки
     - При входе на клетку со спайсом - автоматически собирает его

    3. Червь:
     - После каждого хода игрока появляется на одной из случайных свободных клеток
     - Не может появиться на клетке, занятой харвестером, или на той же клетке, где уже находится
     - Если червь появляется на той же клетке, где находится харвестер - игра проиграна

    4. Победа и поражение:
     - Победа: когда собран весь спайс на поле
     - Поражение: когда игрок попадает в клетку, на которой появляется червь
     - У среднего игрока вероятность выиграть составляет приблизительно 50% при случайных действиях

    5. Счетчики:
     - Количество собранных единиц спайса (из общего количества)
     - Число совершенных ходов

    Технические детали:
     - Используется Canvas 2D API для визуализации
     - Написана на TypeScript с конвертацией в JavaScript
     - Для запуска используется Python HTTP-сервер
     - HTML пользовательский интерфейс с индикаторами состояния
     - Использует объектно-ориентированный подход для реализации игровых сущностей
     - Модульная структура: отдельные файлы для сущностей, игровой логики, отображения, утилит и основного скрипта

    Используемые технологии:
     - Frontend: HTML, CSS, TypeScript
     - Графика: Canvas 2D API
     - Backend: нет (чисто клиентское приложение)
     - Сервер: Python http.server для локального запуска
     - Система сборки: компиляция TypeScript -> JavaScript с помощью tsc (TypeScript Compiler)
     - Визуализация: 2D рендеринг на canvas с использованием цветовой палитры в стиле Dune (коричневый, золотой, бежевые оттенки)
